自由演練 8

Markdown 轉專業報表樣式

把 AI 生成的 Markdown 文字，秒變適合列印給長官的精美報告。不需要 Word，不需要排版——貼進來就完成。

PRACTICE GOALS
學會用 JavaScript 解析 Markdown 轉換為 HTML
理解 CSS 列印樣式（@media print）的設定
產出一個可直接列印為 PDF 的報告工具
TOOL · 互動工具

貼入 Markdown，右側即時渲染為精美報表

左側輸入框支援常用 Markdown 語法：標題（#/##/###）、粗體（**）、斜體（*）、清單（-/1.）、引用（>）、行內代碼（`）。右側即時預覽排版，可直接列印為 PDF。

markdown-to-report.html
左側輸入 Markdown，右側即時預覽排版效果
Markdown 輸入

在左側輸入 Markdown 後即時預覽…

💡
直接從 Gemini 複製貼上。讓 AI 幫你生成報告內容，輸出格式選「Markdown」，複製後貼入左側輸入框，右側立刻出現排版好的報告。
HOW TO REPLICATE · 如何自己做出這個工具

用這段 Prompt，讓 Gemini 幫你做一模一樣的工具

1
打開 Gemini，切換到「Canvas」模式
進入 gemini.google.com，點右上角切換到 Canvas 模式，這樣 AI 生成的 HTML 會直接可以預覽。
2
貼入以下指令
把下方指令完整複製，貼入 Gemini 的輸入框，直接送出。
3
等待生成，預覽並存檔
Gemini 會生成一個完整的 HTML 工具。點「預覽」確認功能正常後，點「下載」或「複製」儲存到你的電腦。
→ 複製這段 Prompt 貼入 Gemini
你是一位擅長製作文件工具的前端工程師。 請製作一個「Markdown 轉專業報表」的單頁 HTML 工具，功能如下： 【功能需求】 - 左右分割版面：左側為 Markdown 文字輸入框（深色背景 code 感），右側即時渲染預覽 - 用 JavaScript 手工解析常用 Markdown 語法：標題 #/##/###、粗體 **、斜體 *、清單 - 和 1.、引用 >、行內代碼 ` - 右側預覽使用精心設計的 CSS：h1 有底線、h2 有分隔線、清單有縮排、blockquote 有左色條 - 頁面右上角有「列印為 PDF」按鈕（window.print()），使用 @media print 隱藏輸入框 【設計要求】 - 背景 #f5f3ee，強調色 #b5703a，h1 底線用強調色 - 預覽區白色背景，字體 Shippori Mincho（標題）+ Noto Sans TC（內文） - 列印時只顯示預覽區內容，隱藏所有工具 UI 請生成完整的單一 HTML 檔案。
CHALLENGE · 進階挑戰

完成基本練習後，試試這些變化

PRACTICE VARIATIONS
01
加入「主題切換」：提供三種 CSS 主題（學術論文風、商務報告風、簡報筆記風），一鍵切換右側樣式。
02
讓工具支援表格語法（| 欄位 | 欄位 |），渲染為帶樣式的 HTML table。
03
加入字數統計與閱讀時間估算（中文每分鐘約 400 字），顯示在預覽區頂部。

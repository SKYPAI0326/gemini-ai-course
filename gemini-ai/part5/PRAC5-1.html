智能電子郵件範本產生器

不用每次從零開始寫信。選好場景範本，填入收件人姓名，一鍵生成完整商務郵件，直接複製貼上。

PRACTICE GOALS
理解如何用 Gemini 生成帶有變數的範本工具
練習「範本選擇 + 欄位填入 + 即時預覽」的互動邏輯
產出一個可實際使用的郵件產生工具
TOOL · 互動工具

選擇場景，填入姓名，一鍵生成郵件

選擇一種郵件場景，填入收件人姓名與公司名稱，下方預覽區會即時顯示完整的商務郵件內文。確認後直接複製貼入你的信箱。

HOW TO REPLICATE · 如何自己做出這個工具

用這段 Prompt，讓 Gemini 幫你做一模一樣的工具

1
打開 Gemini，切換到「Canvas」模式
進入 gemini.google.com，點右上角切換到 Canvas 模式，這樣 AI 生成的 HTML 會直接可以預覽。
2
貼入以下指令
把下方指令完整複製，貼入 Gemini 的輸入框，直接送出。
3
等待生成，預覽並存檔
Gemini 會生成一個完整的 HTML 工具。點「預覽」確認功能正常後，點「下載」或「複製」儲存到你的電腦。
→ 複製這段 Prompt 貼入 Gemini
你是一位擅長製作互動工具的前端工程師。 請製作一個「智能電子郵件範本產生器」的單頁 HTML 工具，功能如下： 【功能需求】 - 頁面頂部有三個按鈕，分別代表三種郵件場景：業務開發、客戶感謝、會議邀請 - 按鈕下方有兩個輸入框：收件人姓名、公司名稱 - 下方有一個預覽區塊，即時顯示填入姓名後的完整郵件內文（使用字串替換） - 預覽區旁有「複製郵件」按鈕，點後複製內容並顯示「已複製！」提示 【設計要求】 - 背景色 #f5f3ee，卡片白色，邊框 #d8d4cb - 強調色 #b5703a（陶土橘），選中的場景按鈕用此色填充 - 字體：標題 Shippori Mincho，內文 Noto Sans TC（Google Fonts） - 預覽區塊用深色背景（#2c2b28）、淺色等寬字體，模擬郵件原文感 - 圓角 6px，無陰影，無漸層 請生成完整的單一 HTML 檔案，包含 CSS 和 JavaScript。
CHALLENGE · 進階挑戰

完成基本練習後，試試這些變化

PRACTICE VARIATIONS
01
在三個現有範本外，新增第四種場景「道歉與補救」，並加入一個「事件說明」的輸入欄位。
02
在 Gemini 生成的工具中，試著加上「正式 / 輕鬆」語氣切換，同一份範本產出兩種版本。
03
把你工作中最常寄的一種郵件場景描述給 Gemini，讓它幫你新增第四個按鈕與對應範本。

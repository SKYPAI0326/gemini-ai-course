自由演練 2

會議記錄行動項目提取牆

把雜亂的會議逐字稿貼進來，工具自動找出所有行動項目，整理成可勾選的待辦清單——再也不用手動逐行掃描。

PRACTICE GOALS
理解如何用 JavaScript 做關鍵字比對與文字分析
練習「輸入 → 處理 → 輸出清單」的工具邏輯
產出一個會議後即用的行動項目整理工具
TOOL · 互動工具

貼入會議記錄，一鍵提取行動項目

將會議逐字稿或筆記貼入下方文字框，工具會自動掃描含有行動關鍵字的句子，整理成可勾選的清單。完成後打勾追蹤進度。

meeting-action-extractor.html
提取結果 · 0 個行動項目
未找到明確的行動項目，試著在文字中包含「請」「需要」「要」「負責」「完成」「確認」等詞彙。
HOW TO REPLICATE · 如何自己做出這個工具

用這段 Prompt，讓 Gemini 幫你做一模一樣的工具

1
打開 Gemini，切換到「Canvas」模式
進入 gemini.google.com，點右上角切換到 Canvas 模式，這樣 AI 生成的 HTML 會直接可以預覽。
2
貼入以下指令
把下方指令完整複製，貼入 Gemini 的輸入框，直接送出。
3
等待生成，預覽並存檔
Gemini 會生成一個完整的 HTML 工具。點「預覽」確認功能正常後，點「下載」或「複製」儲存到你的電腦。
→ 複製這段 Prompt 貼入 Gemini
你是一位擅長製作文字分析工具的前端工程師。 請製作一個「會議記錄行動項目提取器」的單頁 HTML 工具，功能如下： 【功能需求】 - 一個大型文字輸入框（textarea），供使用者貼入會議記錄 - 一個「提取行動項目」按鈕 - 按下後，用 JavaScript 分析文字，找出包含「請」「需要」「要」「負責」「完成」「確認」等行動關鍵字的句子 - 提取結果顯示為可勾選的 checkbox 清單，每一條行動項目為一列 - 若沒有找到行動項目，顯示提示訊息 【設計要求】 - 背景 #f5f3ee，卡片白色，邊框 #d8d4cb，強調色 #b5703a - 字體：標題 Shippori Mincho，內文 Noto Sans TC（Google Fonts） - 勾選框的選中色使用 accent-color: #b5703a - 整體乾淨簡約，圓角 6px，無陰影 請生成完整的單一 HTML 檔案，包含 CSS 和 JavaScript。
CHALLENGE · 進階挑戰

完成基本練習後，試試這些變化

PRACTICE VARIATIONS
01
試著在 Gemini 生成的工具中，加入「截止日期」欄位——讓使用者在行動項目旁邊填入 deadline。
02
擴充關鍵字清單，讓工具能支援英文會議記錄（加入 "need to", "should", "must", "action item" 等）。
03
在提取結果的每一列加一個下拉選單，讓使用者指派負責人（直接輸入姓名）。

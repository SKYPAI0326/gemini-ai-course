PART 3 · 讓數據「動」起來的視覺魔法
3.3

數據儀表板：
實作互動式銷售漏斗

老闆問「這個月成交率多少？」你打開 Excel 截圖給他看——還是直接打開一個會動的漏斗圖，讓他自己移到每個階段，立刻看到轉換率和流失量？同樣的數字，互動呈現的說服力遠比靜態報表強。

學完這節，你會有
一份「互動式銷售漏斗」的完整 Prompt，可直接複製使用
客製化階段數、顏色、標籤的追加指令
加入目標值對比與即時編輯的進階做法
Section 01

為什麼漏斗圖要做成互動版本

銷售漏斗是管理層最常看的報表之一，但大多數公司的漏斗圖還停在 Excel 截圖或 PPT 靜態圖的階段。問題不在於「圖好不好看」，而在於靜態圖無法讓對方自己探索數據。

當你在會議上說「從詢問到提案的轉換率是 27%」，大多數人腦子裡其實沒有畫面。但如果你打開互動漏斗圖，讓他們自己移到那一層，浮動資訊卡立刻顯示「流失了 495 人、轉換 27.2%」——數字馬上有感覺，討論的層次也立刻不一樣。

互動優勢：靜態漏斗只能「看到」結果，互動版本讓每個人在自己有疑問的地方停下來探索——哪個環節流失最嚴重、哪個轉換率異常低，一 hover 就清楚，不需要再翻第二頁報表。
Section 02

互動示範：移到每一層看看

以下是一個真實可互動的銷售漏斗。滑鼠移到任何一層，立刻顯示該階段的人數、轉換率與流失量：

互動式銷售漏斗 · 2024 Q1 業務數據 ↑ 移到各層查看詳細數據
工具解剖 · 互動式銷售漏斗 功能拆解
梯形層 每一層代表一個銷售階段，寬度反映人數比例——越往下人數越少，視覺上自然形成「漏斗」形狀
側標籤 右側顯示各層人數，左側顯示相對上一層的轉換率（第一層為頂端，不顯示轉換率）
Hover 卡 滑鼠移到任何一層時，浮動資訊卡顯示：階段名稱、人數、轉換率、流失量、階段說明
高亮效果 被 hover 的層維持原色，其他層降低透明度，讓視線自然聚焦在當前探索的階段
Section 03

完整 Prompt：互動式銷售漏斗

把以下 Prompt 貼給 Claude 或 ChatGPT，替換掉 【】 內的數字和名稱，就能生成你自己的互動漏斗圖：

互動式銷售漏斗 · 完整 Prompt
你是一位擅長數據視覺化的資深前端工程師。 請幫我用純 HTML + SVG + JavaScript 製作一個「互動式銷售漏斗圖」，規格如下： 【漏斗階段設定】（由寬到窄，共五層） 第一層：廣告曝光｜10,000 人 第二層：點擊訪問｜3,200 人 第三層：詢問留資｜680 人 第四層：提案報價｜185 人 第五層：成交簽約｜42 人 【互動功能】 1. 滑鼠移到任何一層時顯示浮動資訊卡，內容包含： - 階段名稱 - 該層人數 - 與上一層的轉換率（第一層顯示「漏斗頂層」） - 流失人數（上一層 - 本層） - 各層說明文字 2. Hover 的層維持原色；其他層透明度降至 0.35 【視覺規格】 - 梯形顏色：同色系由深到淺，第一層最深 - 右側標示各層人數，左側標示轉換率 - 梯形間留 4px 間距，不要連在一起 - 整體置中，viewBox 自動縮放適應容器寬度 【技術要求】 - 單一 HTML 檔案，不依賴任何外部 library - RWD：手機螢幕自動縮放，資訊卡不超出邊界 - 所有階段資料集中在 JavaScript 陣列方便修改 - 頁面載入時各層依序淡入，有進場動畫
ℹ
替換哪些地方：把五個階段的名稱和人數換成你自己的業務數據即可。如果流程不是五個階段，直接增減層數，AI 會自動重算各層的寬度比例。
Section 04

追加指令：擴充你的漏斗功能

基礎版生成後，用以下追加指令逐步擴充，每條都可以單獨使用：

視覺
把漏斗顏色改為【陶土橘 #b5703a】色系，由深到淺，最底層透明度約 20%
換色系——適合有品牌色指定的場合，也可換成任意 HEX 色碼
功能
每層新增「目標值」欄位，並在梯形上用白色虛線標出目標線；若實際人數低於目標，該層邊框改為橘色警示
目標對比——讓「差多少」一眼看清楚，適合 OKR 或業績追蹤報告
功能
在漏斗上方加「時間篩選器」，可切換本週 / 本月 / 本季，切換時各層數字即時更新並重新繪製
期間篩選——同一份工具可以報告不同週期，不需要重新截圖
進階
在頁面底部加一個輸入表格，讓使用者直接輸入各層人數，漏斗圖即時重新繪製
即時編輯——會議中可以現場改數字，不需要另開 Excel 再截圖貼回
進階
在漏斗右側加「流失排行」區塊，自動計算並排列各階段的流失率，最高的標記為紅色警示標籤
流失排行——快速定位「最需要改善」的環節，直接帶著去開改善會議
★
實用建議：把「即時編輯」追加指令和基礎版合在一起，就是一個輕量版業務儀表板。開會前 5 分鐘填進最新數字，打開瀏覽器直接報告——比 Excel 省時，比 BI 工具省錢，一個 HTML 檔案全搞定。

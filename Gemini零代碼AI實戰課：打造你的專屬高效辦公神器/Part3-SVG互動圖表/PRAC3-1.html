PART 3 · SVG 互動圖表
實例演練 1

技能評估雷達圖

輸入你的技能名稱與分數，即時生成一張 SVG 雷達圖，直觀呈現你的能力分布。最適合個人成長追蹤或團隊能力評估。

PRACTICE GOALS
理解 SVG 圖表的基本座標系與多邊形繪製邏輯
練習操作「輸入數值 → 即時更新圖表」的互動模式
為自己或團隊製作一份真實的技能評估雷達圖
TOOL · 互動工具

填入技能與分數，即時生成雷達圖

在左側填入技能名稱和分數（1–10），右側雷達圖即時更新。你可以新增 3–8 個技能維度，點擊「更新圖表」即可重新渲染。

skill-radar-chart.html
技能名稱 ／ 分數（1–10）
💡
SVG 的核心邏輯：雷達圖是把每個技能的分數轉換成極座標（角度 + 半徑），再用多邊形連接各點。了解這個邏輯，你就能理解為什麼可以用相同的方式做出折線圖、圓餅圖。
HOW TO REPLICATE · 如何自己做出這個工具

讓 Gemini 幫你做一個雷達圖工具

1
想好你的評估維度是什麼
技能評估、績效評估、產品評審……任何需要多維度比較的場景都適合雷達圖。先列出你的 5–8 個維度。
2
複製以下指令貼入 Gemini Canvas
關鍵描述是「用極座標計算 SVG 多邊形的頂點」——這讓 Gemini 知道要用數學計算，而不是靠 CSS 佈局。
3
輸入你自己的技能分數測試
用真實的技能評估數據測試，確認圖形正確渲染，再調整顏色和細節。
→ 複製這段 Prompt 貼入 Gemini
你是一位擅長製作 SVG 數據視覺化工具的前端工程師。 請製作一個「技能評估雷達圖」的單頁 HTML 工具，功能如下： 【功能需求】 - 使用者可輸入 3–8 個技能（技能名稱 + 分數 1–10），支援動態新增和刪除 - 一個「更新雷達圖」按鈕，按下後用 JavaScript 即時渲染 SVG 雷達圖 - SVG 雷達圖包含： · 5 層半透明同心多邊形（分數格線） · 從中心到各頂點的輻射線 · 根據分數計算的實心多邊形（填色半透明 + 描邊） · 每個頂點的技能名稱標籤 · 分數刻度標記（2, 4, 6, 8, 10） - 雷達圖需要是 SVG 格式（不是 Canvas），可縮放 【SVG 計算邏輯】 - 中心點 (cx, cy)，最大半徑 r - 每個技能的角度 = (i / n) * 2π，從頂部（-π/2）開始順時針 - 各頂點座標 = 中心 + (score/10 * r) * (cos(angle), sin(angle)) 【設計要求】 - 整體頁面無印良品風格：背景 #f5f3ee，卡片 #ffffff，邊框 #d8d4cb - SVG 填色使用 rgba(90,122,90,.25)（鼠尾草綠半透明），描邊 #5a7a5a - 格線顏色 rgba(216,212,203,.6)，輔助線 rgba(216,212,203,.4) - 字體：Shippori Mincho + Noto Sans TC（Google Fonts） - 圓角 6px，無陰影，無漸層 請生成完整的單一 HTML 檔案。
CHALLENGE · 進階挑戰

讓雷達圖更有力量

PRACTICE VARIATIONS
01
加入「目標分數」欄位，讓雷達圖可以同時顯示「現況」（實心）和「目標」（虛線輪廓）兩層，清楚看出差距。
02
加入「成員比較」功能——可以輸入兩個人的評分，用不同顏色的多邊形疊加，用於團隊互評或師徒對比。
03
加入「匯出為 PNG」功能，把 SVG 圖表轉換成可分享的圖片。提示：可以讓 Gemini 使用 canvas.toDataURL() 方法。

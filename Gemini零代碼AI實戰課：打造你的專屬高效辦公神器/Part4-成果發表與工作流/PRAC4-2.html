實例演練 2

個人 Prompt 庫管理器

把你在課程中練習的、工作中用過的、覺得好用的 AI 指令，統一存入這個工具，加上標籤和標題，隨時搜尋複製取用。

PRACTICE GOALS
建立你自己的第一個 Prompt 庫，存入至少 5 條指令
理解「新增、搜尋、過濾、複製」的 CRUD 工具邏輯
養成把好用 Prompt 即時記錄的習慣
TOOL · 互動工具

新增、搜尋、一鍵複製你的 Prompt 庫

在下方填入標題、分類標籤和 Prompt 內容，儲存後即可在列表中搜尋和複製。注意：這個工具的資料存在記憶體中，重新整理頁面後會重置，課程結束後你可以用 Gemini 替它加上 localStorage 儲存功能。

prompt-library.html
＋ 新增 Prompt
💡
Prompt 庫的價值不在量，在品。只把真正用過、效果好的 Prompt 存進去，附上清楚的標題，讓自己一眼就知道這條指令是用來做什麼的。這是你最值錢的個人數位資產之一。
HOW TO REPLICATE · 如何自己做出這個工具

讓 Gemini 幫你做一個有永久儲存功能的 Prompt 庫

1
確定你需要哪些欄位
最基本的是標題和內容。如果你有很多種類的 Prompt，加上「分類標籤」可以讓搜尋更快。還需要「使用場景說明」嗎？先想清楚。
2
複製以下指令貼入 Gemini
這個指令特別要求使用 localStorage，讓資料在關掉瀏覽器後也不會消失。
3
存好後立刻把課程中最好的 5 個 Prompt 存入
趁記憶猶新，把這門課程學到的最好用的幾個 Prompt 全存進去，讓這個工具立刻有實質內容。
→ 複製這段 Prompt 貼入 Gemini
你是一位擅長製作互動工具的前端工程師。 請製作一個「個人 Prompt 庫管理器」的單頁 HTML 工具，功能如下： 【功能需求】 - 新增 Prompt：包含「標題」、「分類標籤」（文字）、「Prompt 內容」三個欄位 - 所有資料使用 localStorage 儲存，關閉視窗重開後資料仍在 - 顯示所有已儲存的 Prompt 卡片，每張卡片包含： · 標題（粗體） · 標籤 badge · Prompt 內容（最多顯示 3 行，可點擊展開） · 儲存日期 · 「複製」按鈕、「刪除」按鈕 - 頂部搜尋框：即時過濾（標題 + 內容 關鍵字搜尋） - 標籤過濾器：點選任一標籤可篩選同標籤的 Prompt - 空狀態提示（當庫中無內容時顯示引導文字） 【設計要求】 - 無印良品風格：背景 #f5f3ee，卡片 #ffffff，邊框 #d8d4cb - 強調色 #6b7fa3（霧藍），標籤 badge 用此色系 - 字體：Shippori Mincho + Noto Sans TC（Google Fonts） - 圓角 6px，無陰影，無漸層 請生成完整的單一 HTML 檔案。
CHALLENGE · 進階挑戰

把 Prompt 庫打造成你的知識管理中心
PRACTICE VARIATIONS
01
加入「使用次數計數」功能——每次點複製就 +1，讓最常用的 Prompt 自然排到最上方，建立你的個人熱門榜單。
02
加入「匯出為 JSON」和「匯入 JSON」功能，讓你的 Prompt 庫可以備份到雲端或在不同設備間同步。
03
把課程中 12 個單元最核心的 Prompt（每個單元選 1 個）全部存入，打造你的「課程精華 Prompt 集」，這就是你完課的最大收穫。

MODULE 04

綜合實戰：一站式行動辦公室

課程最終章：n8n 本機指揮中心的完整實戰。雲端傳來指令，n8n 全力執行、AI 生成成果——讓你的電腦成為 24 小時待命的自動化引擎。

學習目標 OUTCOMES
用 Google 表單作為觸發介面，透過 Make 把任務傳送到本機 n8n 執行
理解 n8n 的主場優勢：雲端訊號只是觸發點，本機重型處理才是核心價值
結合 Google AI Studio，實現從一個點子到完整企劃書的全自動產出流程
Remote Control · 4.1

Google 表單遙控：觸發遠端自動化任務

🔀
從 Module 3 到 Module 4 的思維轉換：Module 3 學了兩種本機觸發方式——Watch Folder（新檔落地就跑）和 Schedule（到時間就跑）。兩者都是 n8n 自己決定「什麼時候跑」。Module 4 要學的是另一個維度：由你決定「什麼時候叫它跑」，而且可以在任何地方遠端發出指令。觸發源從本機變成雲端，但重型處理仍由本機 n8n 全力執行。

本節將展示如何將 Google 表單轉變為強大的遠端遙控器。當你在外出差，只需用手機填寫一個 Google 表單，家中電腦的 n8n 立刻開始跑高強度的文件處理任務。這就是「表單 = 遙控器」的概念。

你將學習配置 Google 表單的 Webhook，使其在每次提交時，自動向 n8n 發送數據，進而觸發預先設計好的自動化流程。無論是遠端部署任務、觸發文件生成，還是更新數據，一個表單就能搞定。

📱
Google 表單的優勢：完全免費、不需安裝任何 App、手機電腦均可使用，是最輕量級且易於部署的用戶介面，適合多種遠端觸發情境。
01
Google Forms 遠端遙控 →
Apps Script → Make Webhook → Cloudflare → n8n，六步驟串通整條遙控鏈路。
02
表單數據接收解析 →
掌握 n8n 如何接收並解析 Google 表單傳送的數據，轉化為流程參數。
03
遠端指令控制 →
運用表單數據作為參數，遠端觸發並控制 n8n 自動化流程的執行。
Hybrid Automation · 4.2

Make + n8n 雙引擎：強強聯手

你在上一階段課程學的 Make，與這門課學的 n8n，不是替代關係——而是互補關係。Make 的強項是廣泛串接雲端服務（Gmail、Line、Slack、Airtable），n8n 的強項是在本機執行複雜任務、讀寫本機檔案、無限次跑大量迴圈。兩者結合，你能做到任何一個平台單獨都辦不到的事。

CLOUD
Make.com
  • 廣泛的 SaaS 應用連接（1000+ 服務）
  • 雲端接收 Webhook、觸發事件
  • 視覺化的場景設計
  • 不需要本機環境
LOCAL
n8n Desktop
  • 地端執行，資料不離身
  • 複雜邏輯與大量數據處理
  • 讀取本機檔案系統
  • 完全免費、無次數限制
🤝
協作模式：Make 負責「雲端接收」——接收來自 Gmail、Slack 等雲端服務的事件；n8n 負責「地端處理」——執行高強度文件處理、資料庫操作等本地任務。
01
雙引擎協作實戰 →
n8n 本機處理 → HTTP 回傳 → Make 發 Gmail，完成任務進來到通知發出的完整閉環。
02
Make ↔ n8n API 串接 →
透過 Webhook 或 HTTP API，實現 Make 與 n8n 之間的數據交換與任務傳遞。
03
任務分流協作設計 →
設計雙平台協作模式，將不同類型的任務分配給最適用的平台處理。
Final Project · 4.3

全自動企劃產出：結合 AI 智慧生成

這是整個課程的壓軸實戰。你將整合前三個 Module 的所有技能，結合 Google AI Studio 的 Gemini 模型，打造一條全自動的「點子 → 成品」生產線。

實作任務 FINAL PROJECT
全自動企劃產出流
①用手機填寫 Google 表單，輸入一個活動點子（如「2026 校慶科技展」）
②Make 接收表單資料，觸發 n8n 本機流程
③n8n 將點子傳送至 Google AI Studio（Gemini），AI 生成完整企劃書內容
④自動建立 Google Docs，將企劃書內容填入並排版
⑤同步生成行銷文案、海報文字，存入指定資料夾
⑥完成後自動發送通知（Email / Line），整個流程無需人工介入
🤖
Prompt Engineering 的重要性：精準的提示詞能引導 AI 產出高品質、結構化的企劃內容。我們將學習如何設計能穩定輸出 JSON 格式企劃架構的 Prompt，確保後續節點能正確解析。
01
AI 計劃生成器（壓軸）→
Gemini 生成 JSON 計劃 → Code 解析格式化 → 本機存檔 → Gmail 通知，課程最終實戰。
02
Prompt Engineering →
掌握提示詞工程技巧，引導 AI 產出結構化、可解析的企劃內容。
03
Google Docs 自動輸出 →
自動建立 Google Docs，將 AI 生成的內容填入並格式化，一鍵完成整份企劃書。

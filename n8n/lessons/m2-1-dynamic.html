動態資料引用與 Expression 進階

掌握資料轉換、動態欄位名稱、多筆資料聚合等進階 Expression 技巧，讓流程處理更靈活。

📖參考頁面
⏱閱讀約 10 分鐘
動態欄位名稱（欄位名稱本身是變數）

有時欄位名稱不是固定的，而是由另一個欄位的值決定。例如，你的 JSON 裡有一個 category 欄位，值可能是 "A" 或 "B"，而你要根據這個值去讀對應的欄位 price_A 或 price_B。這時需要在 Code 節點用方括號語法動態存取：

// 在 Code 節點中（不是 Expression） const category = $input.item.json.category; // "A" 或 "B" const fieldName = `price_${category}`; // "price_A" 或 "price_B" const price = $input.item.json[fieldName]; // 動態讀取欄位 return [{ json: { price, category } }];
資料聚合：把多筆 Items 合併成一筆

Loop Over Items 處理完後，每筆 PDF 的結果是獨立的 Item。若要把所有結果合併成一個陣列，可以在 Loop 後接一個 Code 節點：

// 把所有輸入 Items 的 json 合併成一個陣列 const allItems = $input.all(); const results = allItems.map(item => item.json); // 也可以只取特定欄位 const summaries = allItems.map(item => ({ fileName: item.json.fileName, summary: item.json.ai_summary, processedAt: item.json.timestamp })); return [{ json: { results: summaries, total: summaries.length } }];
常用動態引用模式速查
情境解法
根據條件決定讀哪個欄位Code 節點中用 json[動態欄位名]
Loop 後把結果收集成陣列Code 節點用 $input.all().map(i => i.json)
計算所有 Items 的某欄位總和$input.all().reduce((sum, i) => sum + i.json.金額, 0)
篩選出特定條件的 Items$input.all().filter(i => i.json.狀態 === '完成')
找出某欄位最大值的 Item$input.all().sort((a,b) => b.json.金額 - a.json.金額)[0]
💡 Expression vs Code 節點的選擇：單一欄位的讀取、拼接、格式化 → 用 Expression；需要迴圈、陣列操作、複雜邏輯 → 用 Code 節點寫 JavaScript。不要在 Expression 裡硬塞複雜邏輯。

✅ 善用 $input.all() vs $input.item：$input.item 只取當前 Item（單筆），$input.all() 取所有輸入 Items（多筆）。在 Loop 內用 $input.item；在 Loop 後收集用 $input.all()。
